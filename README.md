# Citadel
 Web Project for University of Regina.

👉🏻 FEATURES: 

• Landing Page
• Create Account & Login
• Active User Profile
• Book's Search 
• View, Add, Edit, and Delete Book
• Upload Book Cover
• View Authors Profile
• Live data update
• Error Handling on DOM
• Backend Error Handling
• Admin Login
• Admin Panel to delete users and all the content related to them
• Data storage, and retrieval



👉🏻 SOFTWARE: HTML, CSS, JavaScript, PHP, MySQL

👉🏻 TASKS COMPLETED:

• Preliminary Project Report, Storyboarding, and Layout Sketching had been completed for all web pages.
• Implementation of web pages using HTML and CSS has been completed.

👉🏻 TASKS PLANNED:

• Relational Database Planning and implementation
• Back-end programming
